# Portfolio

Meu portfolio pessoal de projetos, construido com Next.js e TypeScript.

## Rodando localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`. Se a porta estiver ocupada, o Next escolhe outra
porta disponivel e mostra a URL no terminal.

## Scripts

```bash
npm run dev
npm run build
npm run lint
```

## Deploy na Vercel

1. Suba este repositorio para o GitHub.
2. Na Vercel, importe o projeto.
3. Framework preset: `Next.js`.
4. Build command: `npm run build`.
5. Output directory: deixe o padrao da Vercel.
6. Depois do primeiro deploy, adicione o dominio em Project Settings > Domains.

Para dominio comprado fora da Vercel, configure os DNS indicados pela propria
Vercel na tela do dominio.
